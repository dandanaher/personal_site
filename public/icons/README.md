# Icon Assets

Place the following icon PNG files in this directory:

- **me_icon.png** - Clipboard/notepad icon (for Me section)
- **library_icon.png** - Person/user icon (for Library section)
- **projects_icon.png** - Rocket icon (for Projects section)
- **thoughts_icon.png** - Thought bubble icon (for Thoughts section)

## Requirements
- Format: PNG with transparency
- Recommended size: 512x512px or 1024x1024px
- White or light-colored icons work best against the glass orb material
- Icons should be centered in the canvas with padding around edges
